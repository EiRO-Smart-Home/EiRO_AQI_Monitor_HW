# AQI monitor
This project is a developement board based on a MGM260PD32VNA2 Module from SiliconLabs, SEN5x series sensors and with support for ePaper and other SPI based displays.
Other on board components are an SHT41 (Temperature and Humidity), a reset button, a UART to USB bridge and a user controllable LED (D3).

![AQI Board](./media/image.heic)

7 Spare IOs are available on the bottom side of the board and a 0.05" 2x4 connector has the same SPI data lines of the screen with a secondary CS pin (PC07) for other external expansions.
